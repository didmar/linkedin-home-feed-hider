# LinkedIn Home Feed Hider

A Tampermonkey userscript that hides the LinkedIn home feed by default, to reduce distractions.

## Features

- Instantly hides the feed on page load (no flash of content)
- "Show Feed" button appears on the feed page to toggle visibility

## Requirements

This script requires **Tampermonkey**, a browser extension for running userscripts on Chrome, Firefox, Edge, Safari, Opera.

Refer to [the official website](https://www.tampermonkey.net/) for installation instructions.

## Installation

1. Install Tampermonkey for your browser (see above)
2. Click the Tampermonkey icon in your browser toolbar
3. Select "Create a new script..."
4. Delete the default template content
5. Copy the contents of `script.js` and paste it into the editor
6. Press `Ctrl+S` (or `Cmd+S` on Mac) to save
7. Navigate to [linkedin.com/feed](https://www.linkedin.com/feed/)
8. The feed should be hidden, and a button should be visible on the right side to toggle it back
